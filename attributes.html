HTML attributes provide additional information about HTML elements. all html elements contain attributes it provides additional information about element it always specified in start tag like name = "value" some attributes are listed below --- The href attribute of tag specifies the URL of the page the link goes to. The src attribute of tag specifies the path to the image to be displayed. The width and height attributes of tag provide size information for images. The alt attribute of provides an alternate text for an image. The style attribute is used to add styles to an element, such as color, font, size, and more. The lang attribute of the tag declares the language of the Web page The title attribute defines some extra information about an element
